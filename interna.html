Descriçao: Nesta videoaula finalizaremos o nosso curso de CSS. Mas, para considerar o projeto finalizado, devemos criar uma página interna para o nosso projeto. Desta forma procederemos essa criaçao, removendo a regiao de ícones e criando uma regiao destinada ao conteúdo de nossa página. Iremos criar um container para todo o conteúdo, bem como classes que auxiliarao na formataçao de nossas imagens e textos, criando um alinhamento justificado para todos os itens que estiverem dentro de tags do tipo "p" e ainda estiverem dentro de um contexto criado pela classe que define os elementos internos. Ao final, teremos o nosso layout montado, com todos os itens em funcionamento bem como uma página interna para utilizarmos como container de nossos conteúdos internos.
